融合生理模型的血流动力学神经网络

弥合血管分析中的知识鸿沟

传统血管疾病分析方法在统计相关性和机器学习技术之间存在根本的权衡。神经网络具备强大的模式识别能力，但通常需要大量训练数据且可解释性差。融合生理模型的神经网络（PMINNs）提供了一种结合生理模型透明性与机器学习适应性的替代方案。

计算流体力学（CFD）在工业中应用广泛，但血流 CFD 在临床中的影响力因精度限制而未能匹配。HemoDyn 平台通过在生理知识表示与数值精度之间分配计算资源，解决了这一问题。

方法学框架

HemoDyn 平台采用混合架构，集成如下组件：

  • 一维简化血流求解器： 基于 SimVascular 的 ROM 框架高效建模动脉血流
  • 氧气传输模拟： 利用 Elmer 多物理场软件捕捉氧气输送、交换与肌肉消耗动态
  • 穿戴设备数据集成： 实时脉搏与血氧数据提供连续生理输入
  • 机器学习层： 在生理约束下训练的神经网络对不确定性参数建模

该架构的优势包括：

  • 神经网络节点代表生理机制，而非抽象数学函数
  • 预测结果具有可解释性，直接映射至生物过程
  • 结合测量信号与物理结构，显著减少不确定性

实现流程

HemoDyn 的实现流程包括：

  1. 简化下肢血管与肌肉解剖建模
  2. 使用 Gmsh 网格生成并通过 ElmerGrid 转换
  3. 配置 ElmerSolver 模块（ModelPDE、HeatSolve）
  4. 运行 SimVascular ROM 与 ElmerSolver 模拟
  5. 使用 VTK 管线可视化结果
  6. 开发交互控件用于血管参数探索

临床应用与未来方向

HemoDyn 平台对下肢动脉阻塞性疾病（LEAO）具有显著潜力，该病在65岁人群中患病率为12%，在80岁以上人群中达50%。当前临床挑战包括：

  • 难以短期评估治疗干预效果
  • 缺乏量化指标用于疗效评估
  • 个体生理反应差异大

平台与穿戴设备的闭环集成可实现：

  • 持续监测脉搏与血氧
  • 自动采集静息与运动后数据
  • 云端分析与实时治疗趋势反馈

未来发展方向包括：

  • 扩展至慢性病监测应用
  • 与电子健康记录集成以丰富建模
  • 开发药物疗效评估框架
